install Apache:
sudo apt install apache2 -y


Install PHP:
sudo apt install php libapache2-mod-php php-mysql -y


Restart Apache so PHP module loads:
sudo systemctl restart apache2


Create a folder for the project:
sudo mkdir /var/www/html/oilwells


Place index.html and get_wells.php under /var/www/html/oilwells/
Go to http://localhost/oilwells/index.html
